## ðŸƒâ€â™‚ï¸ Getting Started

### Prerequisites

- Node.js >= 18.0.0
- npm or yarn or pnpm

### Installation

1.  **Clone the repository**

    ```bash
    git clone https://github.com/your-username/quiz-it.git
    cd quiz-it
    ```

2.  **Install dependencies**

    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Setup Environment**
    Create a `.env.local` file in the root directory and add your Firebase credentials (see `.env.example`).

4.  **Run the development server**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
